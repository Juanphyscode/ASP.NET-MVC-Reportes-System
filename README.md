# ASP.NET-MVC-Reportes-System
Aplicación web desarrollada en ASP.NET MVC que permite la gestión y visualización de información mediante la integración con una base de datos relacional.

El proyecto incluye el diseño completo de la base de datos, orientado a la administración de personas, facturación y registros asociados, siguiendo un modelo relacional con múltiples entidades y relaciones.

---

## Características principales

* Conexión a base de datos relacional
* Arquitectura basada en el patrón MVC
* Gestión y consulta de información desde interfaz web
* Uso de relaciones mediante claves foráneas
* Estructura escalable para integración de nuevos módulos

---

## Base de datos

El repositorio incluye el script de creación de la base de datos (`.sql`), el cual contiene:

* Definición de tablas
* Claves primarias y foráneas
* Relaciones entre entidades

**Nota:**
El script no incluye datos reales.
El modelo está preparado para ser poblado con datos ficticios o de prueba para su ejecución.
